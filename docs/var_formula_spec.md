# VAR. 시트 셀 ↔ 수식 명세 (var_formula_spec.md)

> 출처: `계산_SHEET_260528_.xlsx` / VAR. 시트 (현재 파일 기준, FIX-01 반영 상태)
> 단위 규약: 길이 mm, 면적 mm², 단면계수 mm³, 단면2차 mm⁴, 힘 N·kN, 모멘트 kN·m
> 이 문서는 계산엔진(FR-CALC) 구현 및 골든 회귀검증의 1차 근거이다.

## 0. 단면 구성 좌표계
- 적층 순서(하단→상단): 후면보드 G→H→I → 스터드 J → 전면보드 K→L→M
- 행3(order): G3·M3=3(외곽), H3·L3=2, I3·K3=1(스터드 인접)
- 빈 레이어는 종류 `----`, 두께 0 (엔진에서는 가변 배열로 대체, `----` 제거)

## 1. 스터드 타입 플래그 (행4~16, C열)
| 셀 | 수식 | 의미 |
|----|------|------|
| C4~C16 | `=IF($A$5=B?,TRUE,FALSE)` | 선택 스터드타입 매칭 플래그 |
| C12 | `=IF($A$5=B12,...)` | 맞댐(2개분) → 폭·면적·Ix ×2 |
| C13 | `=IF($A$5=B13,...)` | 중앙부연결 → 도심 보정 |
| 비고 | C5·C6 = CH/CH개량형 플래그 | **FIX-03 대상**: 중립축 IF 분기에서 TRUE→상수1 처리 |

## 2. 단면 제원 (행5~10)
| 셀 | 수식 | 의미 / 단위 |
|----|------|-------------|
| 행5(높이) | 보드: 입력두께 / 스터드 J5 `=IF(C13,2,1)*VLOOKUP(J4,...,2)+IF(C13,25,0)` | 레이어 높이 mm |
| 행6(폭) | 보드: 스터드간격(C19) 연동 / J6 `=VLOOKUP(...,3)*IF(C12,2,1)` | 폭 mm |
| 행7(유효폭) | `=IFERROR(행35*행6,0)` | 환산유효폭 = n·b |
| 행8(두께) | J8 `=VLOOKUP(...,4)` | 스터드 t |
| 행10(환산단면적) | 보드: `=행7*행5` / J10 `=IF(OR(C12,C13),2,1)*VLOOKUP(...,5)` | A_tr mm² |
| N10 | `=SUM(G10:M10)` | 총 환산단면적 |

## 3. 중립축 (행11~17)
| 셀 | 수식 | 의미 |
|----|------|------|
| 행14(도심거리) | 누적 적층높이 + 자체도심. 스터드 J14는 VLOOKUP(...,7)=cy 사용 | 하단기준 도심 y (mm) |
| 행16 | `=행10*행14` | 환산면적×도심 (1차모멘트) |
| **N15** | `=SUM(G16:M16)/N10` | **중립축 위치 = ΣAy/ΣA** |
| 행17 | `=행14-$N$15` | 중립축 이동거리 d |
| 비고 | J14/K14/L14/M14 의 `IF(OR(C5,C6),1,...)` | **FIX-03**: CH계열 선택 시 적층높이 누락 위험 |

## 4. 단면2차모멘트 (행18~19)
| 셀 | 수식 | 의미 |
|----|------|------|
| 행18(자체I) | 보드: `=1/12*행7*행5^3` / J18 `=VLOOKUP(...,8)*IF(C12,2,1)` | I_self mm⁴ |
| 행19 | `=행10*(행14-$N$15)^2+행18` | 평행축: A·d²+I_self |
| **N19** | `=SUM(G19:M19)` | **완전합성 I_full** |

## 5. 전단연결·합성률 (행20~26)
| 셀 | 수식 | 의미 |
|----|------|------|
| 행20(축파괴강도) | 보드 `=IFERROR(행32*행6*행5*10^-3,0)` / J20 `=J10*J32*10^-3` | 항복축력 kN |
| C31(볼트강도/개) | 엔진 보정: `=0.6*Fy*PI()/4*d^2/1.25` | 볼트 1개 전단강도 N. 기존 Fu 기반식 사용 금지 |
| 행22(볼트전단) | `=$C$31*행21` | 볼트개수×강도 |
| 행23(지압) | `=2*0.85*행5*행21*$C$27*행33` | 지압강도 N |
| 행24(연결전단강도) | `=MIN(행22,행23)*($C$20/2)/볼트간격*10^-3` | 반span 누적 kN |
| 행25(전단강도 누계) | 외곽→내부 `=이전+MIN(행20, 누계행24)` | 부분합성 전단전달 |
| 행26(합성률) | `=행25/J20` 또는 `/MIN(행20,인접20+$J$20)` | **FIX-02**: 분모 비일관 |
| **N26** | `=_xlfn.MINIFS(G26:M26,G26:M26,">0")` | **합성률 η (최소)** |

## 6. 유효강성·휨강도 (행27~36)
| 셀 | 수식 | 의미 |
|----|------|------|
| **N27** | `=J19+SQRT(N26)*(N19-J19)` | **부분합성 I_eff = I_s+√η·(I_f−I_s)** (OI-01) |
| S26 | `=N27*R26` | 보정 I_eff (R26=보정률) |
| 행32(Fy) | 보드 VLOOKUP / J32=275 | 항복강도 N/mm² |
| 행34(E) | 보드 VLOOKUP×10³ / J34=210000 | 탄성계수 N/mm² |
| 행35(n) | `=행34/J34` | 탄성계수비 (보드/스터드) |
| J28/J29(단면계수) | `=J19/J5*2` (= I/(h/2)) | 스터드 S 상·하단 |
| 행36(요소 휨기여) | 보드 `=ABS(MIN(행20,행25)*행17*10^-3)` / J36 `=MAX(|Fy·S|)` | kN·m |
| **N36** | `=SUM(G36:M36)` | **공칭휨강도 Mn** (OI-05) |
