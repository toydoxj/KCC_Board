# VAR. 시트 셀 ↔ 수식 명세 (var_formula_spec.md)

> 출처: `계산_SHEET_260528_.xlsx` / VAR. 시트 (현재 파일 기준, FIX-01 반영 상태)
> 단위 규약: 길이 mm, 면적 mm², 단면계수 mm³, 단면2차 mm⁴, 힘 N·kN, 모멘트 kN·m
> 이 문서는 계산엔진(FR-CALC) 구현 및 골든 회귀검증의 1차 근거이다.

## 0. 단면 구성 좌표계
- 적층 순서(하단→상단): 후면보드 G→H→I → 스터드 J → 전면보드 K→L→M
- 행3(order): G3·M3=3(외곽), H3·L3=2, I3·K3=1(스터드 인접)
- 빈 레이어는 종류 `----`, 두께 0 (엔진에서는 가변 배열로 대체, `----` 제거)

### CH-STUD 후면 끼움 보드 좌표 규칙

- CH-STUD는 후면 석고보드가 CH 홈에 끼워져 고정되는 구조이므로 후면 보드는 1장만 허용한다.
- 일반 CH-STUD의 후면 보드는 25T로 고정한다.
- CH-STUD(개량형)의 후면 보드는 현 자재 DB 기준 12.5T(명목 12T 계열)로 고정한다.
- 중립축 산정 시 후면 보드의 단면적·자체I·도심은 포함하되, 후면 보드 두께는 스터드 및 전면 보드의 적층 좌표에 더하지 않는다.
- 따라서 CH-STUD 스터드 도심은 `y_bottom + 후면보드두께 + cy`가 아니라 `cy`이고, 전면 보드 첫 레이어의 하단 좌표는 `후면보드두께 + H`가 아니라 `H`이다.
- 후면 끼움 보드와 CH-STUD의 합성은 피스 전단연결이 아니라 넓은면 끼움 고정으로 보고 합성률 100%를 적용한다. 엔진에서는 해당 후면 보드의 전단누계를 보드 축강도와 동일하게 둔다.

### I-STUD 후면 고정 보드 좌표 규칙

- I-STUD는 CH-STUD와 동일하게 후면 방화 25T 보드 1장을 붙이고 내부쪽으로 연결하는 형식으로 본다.
- 중립축 산정 시 후면 방화 25T의 단면적·자체I·도심은 포함하되, 후면 보드 두께는 스터드 및 전면 보드의 적층 좌표에 더하지 않는다.
- 따라서 I-STUD 스터드 도심은 `y_bottom + 25 + cy`가 아니라 `cy`이고, 전면 보드 첫 레이어의 하단 좌표는 `25 + H`가 아니라 `H`이다.
- 후면 방화 25T 보드와 I-STUD의 합성률은 100%로 적용한다. 엔진에서는 해당 후면 보드의 전단누계를 보드 축강도와 동일하게 둔다.

### R-STUD 1P 보드-스터드 순 간격 규칙

- R-STUD는 1P 석고보드와 스터드 사이에 12mm의 순 간격이 형성된다.
- 후면 1P 보드가 있으면 후면 보드 적층 끝과 스터드 하단 사이에 12mm를 더한다.
- 전면 1P 보드가 있으면 스터드 상단과 전면 보드 시작 좌표 사이에 12mm를 더한다.
- 이 간격은 단면적이나 질량은 없지만, 각 레이어의 도심거리와 중립축 산정에는 반영한다.

### C-STUD 중앙부 이음 조립 단면

- 스터드 간격: 25mm
- 전체 높이: `H_total = h + 25 + h`
- 단면적: `A_total = 2 × A`
- 스터드 자체 단면2차모멘트 산정 보정계수: 미적용
- 스터드 자체 단면2차모멘트: `Ix = 2 × [A × (h + 25 / 2)^2 + Ix0]`
- 단면계수: `Sx = Ix / h × 2`

## 9. 반력 산정

스터드 간격당 산정된 반력을 `spacing_m`으로 나누어 m당 필요 반력 `kN/m`으로 환산함.

| 항목 | 수식 | 의미 |
|------|------|------|
| Rh,L | `(L * H * B) / 2 / B` (= `L * H / 2`) | 활하중 반력. L=활하중 kN/m², H=검토높이 m, B=스터드간격 m |
| Rh,E | `(Fp / 2 * 2) / B` | 지진하중 반력. Fp는 스터드 간격당 지진력 kN |
| Rh,0.7E | `0.7 * Rh,E` | 지진 조합 반력 |
| Rh,0.75L+0.7E | `0.75 * Rh,L + 0.7 * Rh,E` | 복합 조합 반력 |
| Rh,req | `MAX(Rh,L, Rh,0.7E, Rh,0.75L+0.7E)` | m당 필요 반력 |
| 앵커 성능 | 기본 `0.4 kN/개`, 입력 변경 가능 | 앵커 1개당 저항 성능 |
| 검토 앵커 간격 | 입력값, 기본 `450 mm` | 150~600mm 범위, 50mm 단위 |
| 허용 앵커 간격 | `앵커성능 / Rh,req * 1000` 후 50mm 단위 내림 | 필요 반력 기준 허용 앵커 간격 mm |
| 앵커 사용률 | `Rh,req / (앵커성능 / 검토앵커간격_m)` | 1.0 이하 O.K 기준 |
| 앵커 기준 최대높이 | 50mm 단위 탐색 | 응력·처짐·앵커 사용률을 모두 만족하는 최대 높이 |
