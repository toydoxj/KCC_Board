# Property_Gymsumboard 반영 현황

`계산_SHEET_260528_.xlsx / Property_Gymsumboard` 출처의 보드 물성은 `data/seed/board_property.json`에 22개 조합으로 반영되어 있다.

계산엔진에서 사용하려면 `mass_kg_m2`, `Fu`, `E_GPa`가 필요하다. `Fu`는 `Property_Gymsumboard`의 `파괴응력 검토` 표를 적용하고, 지압강도 `Fb` 산정에 직접 사용한다. 보드 항복강도 `Fy`는 `0.6 * Fu`로 산정한다. `E_GPa`는 사용자 제공 표를 적용하며, 값이 누락된 경우에만 기본값 `1.9 GPa`를 적용한다. 레거시 DB에서 `Fu`가 비어 있으면 저장된 `Fy`를 레거시 항복강도로 보고 `Fu = Fy / 0.6`으로 역산한다.
