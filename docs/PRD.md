|------|------|

| 버전 | v0.1 (초안) |

## 1. 개요

### 1.1 배경
- 기존 검토는 다중 시트·다중 VLOOKUP·셀 좌표 종속 수식으로 구성된 엑셀시트로 수행되고 있음.
- 엑셀 구조상 (1) 입력 변경 시 셀 좌표 IF 분기 오류 위험, (2) 다중 케이스 비교 곤란, (3) 계산서 수기 정리 부담, (4) 단면 선정의 반복 시행착오가 존재함.
- 동일한 역학 로직을 웹 프로그램으로 이관하여 입력 검증·자동 단면선정·계산서 자동출력을 제공하고자 함.

### 1.2 목적
- 경량 스터드와 석고보드(전·후면)로 구성된 건식벽체의 **면외 부분합성 단면 거동**을 검토하는 엔진을 구현함.
- 엔지니어가 벽체 구성·하중·지간을 입력하면 **검토(휨·처짐) 판정**, **만족 단면 자동선정**, **계산서 자동생성**을 수행함.

### 1.3 범위 (In-Scope)
- 면외 단순지지 1방향 부재(스터드)에 대한 휨·처짐·지진 검토
- 부분합성 단면(환산단면·합성률·유효강성) 산정
- 스터드/석고보드/볼트 자재 DB 관리
- 단면 자동선정·최적화 및 계산서(PDF/문서) 자동생성

### 1.4 비범위 (Out-of-Scope)
- 면내(축·전단벽) 거동, 좌굴 상세, 접합부 상세설계
- 3D 해석, 동적 시간이력 해석
- 기존 엑셀의 SCREWBOLT 시트 볼트 강도 산정식 자체의 신규 정식화(엑셀 로직 이관에 한정하되, `#REF!`는 정상화 — §6)

> **불확실 표기:** 본 PRD의 역학 정식(부분합성 √η, 지진 집중하중 등)은 엑셀 로직을 그대로 이관함을 전제로 함. 적용 KDS/AISC 조항 번호는 **확인 필요**로 표기한 항목을 개발 착수 전 확정해야 함.

## 2. 도메인 배경 (검토 이론 요약)

본 프로그램이 재현할 핵심 역학 흐름이다. 모든 식은 엑셀 VAR. 시트에서 검증된 거동을 기준으로 함.

| 단계 | 개념 | 엑셀 근거(VAR.) | 비고 |
|------|------|------------------|------|
| ① 환산단면 | 보드를 탄성계수비 n으로 폭 환산 | 행7(유효폭), 행35(n) | n = E_board / E_stud |
| ② 중립축 | 환산 1차모멘트/환산면적 | N15 = ΣAy/ΣA | 평행축 기준점 |
| ③ 단면2차모멘트 | 평행축 정리 합산 | 행19 = A·d² + I_self, N19 | 완전합성 가정값 |
| ④ 전단연결 | 볼트 전단·지압의 작은 값, 외곽부터 누계 | 행22~25 | 부분합성 전단전달 |
| ⑤ 합성률 η | 전달 전단력/완전합성 필요력 | 행26, N26=MINIFS(>0) | 0≤η≤1 |
| ⑥ 유효강성 | 부분합성 원값 산정 후 I_eff 보정계수 적용 | N27(raw) = I_s + √η·(I_full − I_s), I_eff = N27(raw)×R | AISC형 √η — **출처 확인 필요** |
| ⑦ 휨강도 Mn | 보드 기여 + 스터드 항복모멘트 합 | N36 = ΣG36:M36 | 요소별 합산 |
| ⑧ 처짐 | 등분포 단순보 5wL⁴/384EI | C22 | 수평하중(24kg/m²) 기준, 지진 제외 |
| ⑨ 지진력 | 비구조요소 수평지진력(집중하중) | C40~C43 | KDS 41 17 00 — **조항 확인 필요** |
| ⑩ 하중조합 | D / D+L / E 포함 조합 | 행45~49 | C48 지배 |
| ⑪ 반력·앵커 | L, 0.7E, 0.75L+0.7E 중 최대값을 m당 필요 반력으로 환산 후 앵커 간격 산정 | Rh,req, s_anchor | 기본 앵커 성능 0.4kN/개 |
| ⑫ 판정 | 응력비 ΩMu/Mn≤1, 처짐 δ≤L/240 | C21·C24·C25 | ASD, Ω=1.67 |

**확정된 설계 의도(사용자 확인 완료):**
- 24 kg/m² 는 **수평하중**(활하중과 동일 성격)이며 처짐·휨에 일관 적용함.
- 지진하중은 **집중하중**으로 고려함 → 면외 모멘트 계수 1/4 적용은 의도된 값임.
- 처짐 검토에서 지진하중은 **제외**함.

## 3. 사용자 및 핵심 시나리오

### 3.1 페르소나
| 페르소나 | 설명 | 핵심 니즈 |
|----------|------|-----------|
| 구조 검토자(주) | 건식벽체 구조검토 수행 엔지니어 | 빠른 판정, 자동 단면선정, 계산서 |
| 검토 보조자 | 입력·케이스 작성 담당 | 입력 검증, 자재 DB 선택 |

### 3.2 핵심 사용 시나리오
1. **단일 검토:** 벽체 구성·하중·지간 입력 → 즉시 판정(O.K/N.G) 및 상세값 확인.
2. **단면 자동선정(핵심):** 하중·지간·제한조건 고정 → 만족하는 최소 단면구성(스터드 규격·보드 구성·스터드 간격·볼트 배치)을 자동 탐색.
3. **계산서 생성(핵심):** 확정된 케이스를 계산서(서식 포함)로 출력하여 로컬에 저장(Electron 저장 대화상자).
4. **케이스 비교:** 복수 구성안의 판정·여유율을 표로 비교(선택 기능).

## 5. 기능 요구사항 (Functional Requirements)

### 5.1 입력 관리 — 벽체 구성 모델 (FR-IN)

엑셀의 G~M열 고정 적층을 **레이어 배열 모델**로 일반화함.

| ID | 요구사항 |
|----|----------|
| FR-IN-01 | 벽체를 후면보드 레이어[ ] + 스터드 + 전면보드 레이어[ ] 적층으로 입력함 |
| FR-IN-02 | 각 보드 레이어 = {종류(일반/방화/방수/차음/…), 두께(9.5/12.5/15/19/25mm)} |
| FR-IN-03 | 스터드 = {타입(C/CH/HR/I/MP/RV/R/T.silent), 규격, 시공방식(기본/겹침/맞댐/중앙부 이음/이중)} |
| FR-IN-03A | C-STUD 중앙부 이음은 스터드 2개를 25mm 이격한 조립 단면으로 산정하고, 스터드 자체 단면2차모멘트는 보정계수 없이 `2 × [A × (h + 25 / 2)^2 + Ix0]`를 적용함 |
| FR-IN-03B | CH-STUD는 후면 석고보드 1장만 CH 홈에 끼움 설치하며 일반형은 25T, 개량형은 현 DB 기준 12.5T(명목 12T 계열)로 고정함. 중립축 좌표 산정 시 이 후면 보드 두께는 스터드·전면 보드 적층 좌표에서 제외하고, 후면 끼움 보드와 스터드의 합성률은 100%로 적용함 |
| FR-IN-03C | R-STUD는 1P 석고보드와 스터드 사이에 12mm 순 간격이 형성되므로, 실제 1P 보드가 있는 쪽의 보드-스터드 사이 좌표에 각각 12mm 간격을 반영함 |
| FR-IN-03D | I-STUD는 CH-STUD와 동일한 후면 고정 보드 형식으로 산정하되, 후면 보드는 방화 25T 1장으로 고정함. 중립축 좌표 산정 시 후면 보드 두께는 스터드·전면 보드 적층 좌표에서 제외하고, 후면 보드와 스터드의 합성률은 100%로 적용함 |
| FR-IN-04 | 하중: 수평하중(kg/m², 기본 24), 지진 파라미터(S, 지반등급 S1~S5 기반 자동 Fa, 중요도계수 Ip) |
| FR-IN-05 | 기하: 스터드 간격(mm), 지지길이 L(mm), 처짐한계(1/240 등) |
| FR-IN-06 | 볼트: 지름, 항복강도(Fy, UI 숨김 기본값), 간격(1P/2P/3P), 연결볼트 개수(1P/2P/3P 개별 입력) |
| FR-IN-07 | 입력 검증: 미존재 보드 조합·음수·DB 미등록 규격 차단(엑셀 #N/A 원천 제거) |

### 5.3 단면검토 계산엔진 (FR-CALC) — 핵심

엑셀 VAR. 계산순서를 **순수함수 파이프라인**으로 구현함(pure functions + Repository/DI).

| ID | 계산 단계 | 입·출력 | 엑셀 매핑 |
|----|-----------|---------|-----------|
| FR-CALC-01 | 탄성계수비 n | E_board/E_stud | 행34·35 |
| FR-CALC-02 | 환산유효폭·환산단면적 | n·b, A_tr | 행7·10 |
| FR-CALC-03 | 단위중량·총중량 | kN/m, kN | 행12·13 |
| FR-CALC-04 | 중립축 위치 | ΣAy/ΣA | 행14·16, N15 |
| FR-CALC-05 | 단면2차(완전합성) | A·d²+I_self | 행18·19, N19 |
| FR-CALC-06 | 볼트 전단·지압강도 | min(0.5·0.6·0.85·Fu_bolt 기반 전단, Fu_board 기반 지압) | 행22·23·24 |
| FR-CALC-07 | 전단연결 누계·합성률 η | 외곽→내부 누계 | 행25·26, N26 |
| FR-CALC-08 | 유효단면2차 I_eff | I_eff(raw)=I_s+√η·(I_f−I_s), I_eff=I_eff(raw)×보정계수 | N27/S26 |
| FR-CALC-09 | 공칭휨강도 Mn | 보드기여+스터드 Fy·S | 행36, N36 |
| FR-CALC-10 | 처짐 δ | 5wL⁴/384EI (지진제외) | C22 |
| FR-CALC-11 | 지진력 Fp·지진모멘트 | 집중하중(1/4) | C40~C43 |
| FR-CALC-12 | 하중조합·소요휨 Mu | D/D+L/E max | 행45~49, C3 |
| FR-CALC-13 | 판정 | ΩMu/Mn≤1, δ≤L/c | C21·C24·C25 |
| FR-CALC-14 | 최대 가능 높이 | 50mm 단위로 L을 증분 검토하여 응력·처짐 모두 O.K인 최대값 | 엔진 확장 |
| FR-CALC-15 | 전 단계 중간값 반환(계산서·검증용) | JSON | — |
| FR-CALC-16 | 반력 및 앵커 간격 산정 | Rh,L=(L·H·B)/2/B, Rh,E=(Fp/2·2)/B, Rh=max(L,0.7E,0.75L+0.7E), 앵커간격=앵커성능/Rh×1000 | 엔진 확장 |

- 모든 계산은 단위 명시 후 내부 단위계 일관 유지함.
- Ω(=1.67)·처짐한계는 설정값 분리(하드코딩 금지).
- I_eff 보정계수는 최종 `I_eff`에만 적용함. 계수는 C-STUD 1.0, CH-STUD/개량형 0.58, T-Silent 0.44, R-STUD 0.28, I-STUD 0.8, HR-STUD 0.55, RV-STUD 0.45, C-STUD(맞댐) 1.0, C-STUD(중앙부연결/중앙부 이음) 0.22, MP-STUD 0.45로 함.
- 엑셀 **회귀검증 테스트** 필수 포함(§7).

### 15.3 산정·검증 근거
- 면적: Shoelace 공식(외곽 폐곡선 직접 적분)

- rx·ry: √(I/A)
- 데이터 소스: `stud.dxf` (AutoCAD 2018) → 폴리라인 추출 자동 산정
- 단면 형상 참조: `단면_시각화_2.png` (그룹별 규격 형상)

> **개선 메모:** 엔진은 VLOOKUP 인덱스 의존을 제거하고 **그룹+단면명 키 기반** 조회로 구현함. section_class는 환산/유효폭 처리 분기(박판·합단면)에 활용함.
