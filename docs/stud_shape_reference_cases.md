# 스터드 형상별 기준 계산값

작성일: 2026-07-24 (나사 전단식·제원 및 I_eff 보정계수(실험 보정률) 갱신 반영)

## 목적

아래 표는 요청된 11개 벽체 형상에 대해 현재 계산 엔진으로 산출한 기준값이다. 프로그램 수정 후 값 변화를 확인할 수 있도록 동일한 입력과 기대값을 `tests/golden/requested_shape_cases.json`에 저장했고, `tests/test_requested_shape_cases.py`에서 반복 검증한다.

## 산정 근거 변경 (2026-07-24)

- 커밋 `2e5f729` "나사(피스) 전단강도 산정식·제원 갱신" 기준으로 전 표를 재산정했다.
- 피스 1개당 전단강도 식을 `0.5 × Fv × Ab / 1.25` (Fv = 0.6 × Fy)로 변경하고 기존 0.85 계수를 제거했다. (EN 1993-1-8 계열, `var_formula_spec.md` §5 C31)
- 피스 제원을 Fu 480 기준에서 Fy 340 기준으로 교체하여 피스 1개 전단강도가 **942.1 N → 785.1 N**으로 감소했다.
- 전단연결력 감소로 합성률·유효강성·공칭휨강도가 전반적으로 하향되어, 11종 모두 최대 높이·Mu·Ieff가 이전 값 이하로 조정되었다(I_eff 보정계수는 불변).
- 엑셀 정본은 폐기되었으며, 엔진 출력을 골든 기준으로 한다.
- (2026-07-24 추가) 실험 기반 I_eff 보정계수(실험 보정률)를 갱신했다: C-STUD 중앙부이음 0.23→0.25, CH-STUD/개량형 0.62→0.65, HR-STUD 0.67→0.71, MP-STUD 0.38→0.42, RV-STUD 0.40→0.43, T-Silent 0.38→0.40, R-STUD 0.24→0.26, I-STUD 0.85→0.87 (C-STUD 기본/맞댐 0.7·후면 없음 1.0 불변). 보정계수는 최종 `I_eff`에만 곱하므로 Ieff raw(보정 전)는 불변이며 최대 높이·Mu가 재산정되었다.

## 공통 입력

- 설계 CASE: seismic
- 강도 검토: composite
- 수평하중: 0.25 kN/m2 (= 25.48 kg/m2)
- 처짐 한계: L/240
- 볼트(피스): 직경 3.5 mm, **Fy 340 N/mm2** (전단강도 Fv = 0.6 × Fy), 피치 600/300/300 mm, 개수 2개
- 지진 입력: S 0.22, Ip 1.5, Fa 1.3
- 앵커: 0.4 kN, 450 mm
- 기본 보드: 방화 19T 2P 양면
- 최대 높이: 계산 엔진의 `max_height_mm` 값이며, 앵커 최대 높이는 제외
- Ieff: 2400 mm 높이에서 `I_eff_raw_mm4` 값, 즉 보정계수 적용 전 값

## 특이 사항

- 2번은 전면만 방화 19T 2P이며 후면 보드는 없다.
- 5번과 11번은 후면 방화 25T 1P, 전면 방화 15T 3P로 계산했다.
- 7번 사용자 표기 `MP-STUD HR-74`는 자재 DB에 `HR-74`가 없어 `MP-STUD MP-74`로 계산했다.
- 9번 T.silent-STUD는 간격이 별도 기재되지 않아 450 mm로 계산했다.
- C-STUD 맞댐이음은 엔진 로직상 스터드 2개로, 중앙부 이음은 25 mm 간격을 둔 스터드 2개로 계산된다.
- C-STUD 기본은 후면 보드가 없으면 R=1.0, 후면 보드가 있으면 R=0.7을 적용한다.
- 나사 전단강도 하향으로 3번(C-STUD 맞댐이음)은 처짐 판정이 O.K → N.G로 변경되었다(δ 31.22 → 33.65 mm > 한계 31.25 mm).
