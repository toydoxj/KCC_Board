# 스터드 형상별 기준 계산값

작성일: 2026-07-07

## 목적

아래 표는 요청된 11개 벽체 형상에 대해 현재 계산 엔진으로 산출한 기준값이다. 프로그램 수정 후 값 변화를 확인할 수 있도록 동일한 입력과 기대값을 `tests/golden/requested_shape_cases.json`에 저장했고, `tests/test_requested_shape_cases.py`에서 반복 검증한다.

## 공통 입력

- 설계 CASE: seismic
- 강도 검토: composite
- 수평하중: 0.25 kN/m2
- 처짐 한계: L/240
- 볼트: 직경 3.5 mm, Fy 480 N/mm2, 피치 600/300/300 mm, 개수 2개
- 지진 입력: S 0.22, Ip 1.5, Fa 1.3
- 앵커: 0.4 kN, 450 mm
- 기본 보드: 방화 19T 2P 양면
- 최대 높이: 계산 엔진의 `max_height_mm` 값이며, 앵커 최대 높이는 제외
- Ieff: 2400 mm 높이에서 `I_eff_raw_mm4` 값, 즉 보정계수 적용 전 값

## 특이 사항

- 2번은 전면만 방화 19T 2P이며 후면 보드는 없다.
- 5번과 11번은 후면 방화 25T 1P, 전면 방화 15T 3P로 계산했다.
- 7번 사용자 표기 `MP-STUD HR-74`는 자재 DB에 `HR-74`가 없어 `MP-STUD MP-74`로 계산했다.
- 9번 T.silent-STUD는 간격이 별도 기재되지 않아 450 mm로 계산했다.
- C-STUD 맞댐이음은 엔진 로직상 스터드 2개로, 중앙부 이음은 25 mm 간격을 둔 스터드 2개로 계산된다.
- C-STUD 기본은 후면 보드가 없으면 R=1.0, 후면 보드가 있으면 R=0.7을 적용한다.
